# 图书推荐系统 python103

## 项目概述


## 技术栈

- **后端**: Python
- **人脸识别库**: dlib
- **图像处理**: OpenCV
- **前端**: PyQt5
- **数据库**: MySQL

## 功能模块

- 用户注册与登录
- 人脸识别登录
- 图书检索
- 个性化推荐
- 阅读记录管理

## 系统角色

- **普通用户**: 注册、登录、检索图书、查看推荐、管理阅读记录
- **管理员**: 用户管理、图书管理、系统维护

## 运行环境

- **开发环境**: Pycharm
- **Python版本**: 3.8
- **依赖库**: PyQt5、OpenCV

## 部署步骤

1. 环境准备：安装Python3.8、PyQt5、OpenCV等相关依赖库
2. 数据库搭建：配置MySQL数据库
3. 系统部署：运行主程序，启动图书推荐系统

## 目录结构

```
图书推荐系统
├── backend              # 后端代码目录
├── frontend             # 前端代码目录
├── resources            # 资源文件目录
└── database             # 数据库文件目录
```

## 核心亮点

- **智能化推荐**: 基于用户阅读历史，结合机器学习算法，实现个性化图书推荐
- **跨平台**: 前端使用PyQt5，支持多平台运行，具有良好的兼容性
- **易于维护**: 清晰的代码结构，便于系统维护与功能扩展
